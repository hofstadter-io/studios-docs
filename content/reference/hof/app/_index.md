#### hof app

```Work with your Studios App

Usage:
  hof app [command]

Aliases:
  app, a

Available Commands:
  available-versions Get the runtime version of your App
  create             
  delete             
  deploy             Deploys the App
  generate           Validate your application
  list               List app of your Apps
  pull               Get the latest version from Studios
  push               Send and make the latest version on Studios
  reset              Reset the App
  secrets            Set the App Secrets
  shutdown           Shutdowns the App
  status             Get the status of your App
  update             Updates the Application runtime
  validate           Validate your application
  version            Get the runtime version of your App

Flags:
  -h, --help   help for app

Global Flags:
  -c, --context string   the context to use for the hof tool and commands

Use "hof app [command] --help" for more information about a command.
```
