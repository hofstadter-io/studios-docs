#### hof config

```Configure the Hof CLI

Usage:
  hof config [command]

Available Commands:
  get         Get Hofsadter configuration(s)
  set         Get configuration values
  set-context Set a context configuration
  test        Test a context configuration
  use-context Use a context configuration

Flags:
  -h, --help   help for config

Global Flags:
  -c, --context string   the context to use for the hof tool and commands

Use "hof config [command] --help" for more information about a command.
```
