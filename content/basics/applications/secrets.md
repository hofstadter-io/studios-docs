The __secrets__ directory holds the sensative
information for your application.

```
app:
  name: "hof-starter-app"

  secrets:
    mailgun:
      apikey: "..."
      domain: "..."
    oauth:
      google:
        app_id: "..."
        apikey: "..."
```

__note, this file format and layout is still subject to change.__
