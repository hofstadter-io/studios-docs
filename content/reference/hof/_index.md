#### hof

```Hofstadter Studios is a platform
for collaborative development.

Usage:
  hof [command]

Available Commands:
  app             Work with your Studios App
  bash-completion Generate  Bash completions
  config          Configure the Hof CLI
  db              Work with your Studios DB
  function        Work with your Studios Functions
  help            Help about any command
  import          Import bundles for types, modules, pages, packages and components.
  new             Make new apps, types, modules, pages, and components.

Flags:
  -c, --context string   the context to use for the hof tool and commands
  -h, --help             help for hof

Use "hof [command] --help" for more information about a command.
```
