#### hof config set

```Get configuration values

Usage:
  hof config set [command]

Available Commands:
  account     Set your account ID
  apikey      Set your API Key
  host        Set your host server

Flags:
  -h, --help   help for set

Global Flags:
  -c, --context string   the context to use for the hof tool and commands

Use "hof config set [command] --help" for more information about a command.
```
