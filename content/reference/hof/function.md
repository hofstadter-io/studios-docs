#### hof function

```Work with your Studios Functions

Usage:
  hof function [command]

Aliases:
  function, functions, funcs, func, fns, fn

Available Commands:
  call        Call a function by name
  delete      Deletes the function <name>
  deploy      Deploys the function <name>
  pull        Get the latest version from Studios
  push        Send and make the latest version on Studios
  status      Get the status of your functions

Flags:
  -h, --help   help for function

Global Flags:
  -c, --context string   the context to use for the hof tool and commands

Use "hof function [command] --help" for more information about a command.
```
